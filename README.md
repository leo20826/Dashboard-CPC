# Dashboard-CPC

Dashboard desarrollado en React para monitorear el progreso de estudiantes en diferentes jueces en línea, siguiendo el estilo de competencias ICPC.

## Descripción

Este proyecto proporciona una plataforma visual y corporativa para que entrenadores y coordinadores puedan supervisar el desempeño y avance de sus estudiantes en plataformas de programación competitiva (jueces en línea) con un enfoque en el formato ICPC. El dashboard se conecta a APIs de jueces y utiliza Google Sheets para gestionar la lista de estudiantes y sus usuarios en cada plataforma.

## Características principales

- Monitoreo del progreso de estudiantes en jueces en línea.
- Integración con APIs de jueces estilo ICPC.
- Lectura dinámica de la lista de estudiantes desde Google Sheets.
- Interfaz moderna y estilo corporativo adaptado a competencias ICPC.
- Código modular y fácil de extender para agregar más jueces o funcionalidades.

## Tecnologías

- React.js  
- APIs de jueces en línea (Codeforces, VJudge, etc.)  
- Google Sheets API para gestión de estudiantes  
- Librerías para visualización de datos (por ejemplo, Chart.js)
